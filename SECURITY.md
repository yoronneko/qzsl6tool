# Security Policy

## Supported Versions

qzsl6tool currently uses `0.x` semantic versions. Security updates are only
provided for the latest release in the current `0.1.x` series.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |
